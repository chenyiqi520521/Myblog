涵盖一个“正经”的资讯类app开发过程中遇到的点点滴滴  
适合新手入门，在这里你可以：  
1. 接触到最新的RecyclerView、CardView  
2. 了解到日间、夜间模式如何实现：配色法、蒙板法  
3. 如何实现自己的一个图片功能模块，提供了L1内存缓存 L2磁盘缓存 L3网络读取  
4. 如何在webView里面针对图片，实现自适应手机屏幕宽度、点击在Activity查看大图、wifi下有图3g下无图。  
5. 如何自己实现通用的图片、存储模块。
